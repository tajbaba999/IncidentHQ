# Interview Prep Prompts

## Combined DSA + LLD Prompt

```
Solve this problem for a tech interview. 

For DSA Problems:
1. Explain the approach/algorithm
2. Provide time and space complexity (Big O)
3. Write complete Java code
4. List edge cases
5. Suggest follow-up optimizations

For System Design/LLD Problems:
1. Requirements Analysis (functional + non-functional)
2. Entity definitions with all fields, types, and relationships
3. Class diagram with responsibilities and design patterns
4. Database schema (tables, indexes, foreign keys)
5. API endpoints with request/response formats
6. Key design decisions and trade-offs
7. Scalability considerations
8. Core Java implementation

Problem: [PASTE YOUR QUESTION HERE]
```

---

## Quick Copy Prompts

### DSA Only
```
Solve in Java: [problem]. Show approach, O complexity, code, edge cases, and follow-ups.
```

### LLD Only
```
Design [system] in Java. Include: entities with all fields, class diagram, database schema, API design, design patterns, and core code.
```

---

## Entity Description Template (for LLD)

When asked "tell me about the entity", answer:
```
Entity: [Name]
- Fields: [list all with type and constraints]
- Relationships: [one-to-one, one-to-many, etc.]
- Validations: [not null, unique, etc.]
- Example: [sample data]
```

---

## Example Usage

**DSA:**
```
Solve in Java: Given an array of integers and a target sum, find all unique triplets that add up to the target.
```

**LLD:**
```
Design a parking lot system in Java. Include entities (Vehicle, ParkingSlot, Ticket), classes, database schema, and core implementation.
```
